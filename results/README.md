# Results

This folder contains saved model outputs from the forecasting experiment.

- `results_table.csv`: model comparison table with MSFE, MAE, and relative MSFE.
- `forecasts_h1.csv`: one-quarter-ahead forecasts.
- `forecasts_h4.csv`: four-quarter-ahead forecasts.
- `errors_h1.csv`: one-quarter-ahead forecast errors.
- `errors_h4.csv`: four-quarter-ahead forecast errors.
- `summary.json`: project metadata, predictors, factor information, and top Lasso-selected variables.
